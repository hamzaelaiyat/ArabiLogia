# عربيلوجيا (ArabiLogia)

تطبيق Flutter للهاتف المحمول لتعلم اللغة العربية، يتضمن تمارين في النحو، والصرف، والأدب، والشعر، وفهم المقروء لطلاب المرحلة الثانوية (الصفوف 10-12).

## المميزات

- **المصادقة** - تسجيل الدخول وتسجيل الخروج بالبريد الإلكتروني وكلمة المرور، اختيار الصف الدراسي، الموافقة على الشروط
- **تصنيفات الاختبارات**:
  - النحو
  - الصرف
  - الأدب
  - الشعر
  - القراءة والنصوص
- **لوحة التحكم** - الصفحة الرئيسية، المهام، لوحة المتصدرين، الملف الشخصي
- **نظام الاختبارات** - اختبارات تفاعلية مع أنماط أسئلة متعددة، المؤقت، النتائج الفورية
- **لوحة الإدارة** - إدارة الاختبارات للأساتذة والإدارة، محرر الأسئلة، مدير النصوص، عارض النتائج
- **سجل النشاط** - تتبع الاختبارات المنجزة والأداء عبر الزمن
- **الإعدادات** - تخصيص المظهر (فاتح/داكن/النظام)، الإشعارات، الخصوصية
- **واجهة ثنائية اللغة** - واجهة عربية بالكامل مع دعم RTL
- **مميزات الهاتف** - الإشعارات_push، ومشاركة النتائج، والتحديثات داخل التطبيق

## التقنية المستخدمة

- **الإطار**: Flutter 3.x
- **اللغة**: Dart 3.x
- **إدارة الحالة**: Provider
- **التوجيه**: GoRouter
- **الخادم الخلفي**: Supabase (المصادقة، قاعدة البيانات)
- **الإعلانات**: Google Mobile Ads
- **الهندسة**: هندسة نظيفة مبنية على الميزات

## البدء

### المتطلبات المسبقة

- Flutter SDK 3.x
- Dart 3.x
- حساب Supabase

### التثبيت

1. استنساخ المستودع:
   ```bash
   git clone https://github.com/your-org/ArabiLogia.git
   cd ArabiLogia
   ```

2. تثبيت التبعيات:
   ```bash
   flutter pub get
   ```

3. إعداد Supabase:
   - إنشاء مشروع Supabase
   - نسخ `.env.example` إلى `.env`:
     ```bash
     cp .env.example .env
     ```
   - تحديث `.env` ببيانات اعتماد Supabase:
     ```
     SUPABASE_URL=https://your-project.supabase.co
     SUPABASE_ANON_KEY=your-anon-key
     ```

4. تشغيل التطبيق:
   ```bash
   flutter run
   ```

### البناء للإصدار

```bash
flutter build apk --release
flutter build ipa --release  # iOS (يتطلب macOS)
```

## هيكل المشروع

```
lib/
├── core/
│   ├── config/         # إعداد Supabase
│   ├── constants/      # سلاسل التطبيق، المسارات، المحتوى القانوني
│   ├── routes/        # توجيه التطبيق (GoRouter)
│   ├── services/      # خدمات API (Supabase، الإشعارات، إلخ)
│   ├── theme/         # إعداد المظهر (فاتح/داكن/الرموز)
│   └── widgets/       # المكونات المشتركة (زجاج، إعلانات أصلية)
├── features/
│   ├── admin/        # لوحة الأساتذة والإدارة (محرر الاختبارات، النتائج، الإعدادات)
│   ├── auth/         # تسجيل الدخول، تسجيل الخروج، نسيت كلمة المرور
│   ├── dashboard/    # التطبيق الرئيسي (الرئيسية، الاختبارات، لوحة المتصدرين، الملف الشخصي، السجل)
│   └── legal/        # المحتوى القانوني وأوراق البيانات
├── providers/        # إدارة حالة Riverpod/Provider
└── main.dart        # نقطة دخول التطبيق
```
