# عربيلوجيا (ArabiLogia)

تطبيق Flutter لتعلم اللغة العربية، يتضمن القواعد، الصرف، الأدب، الشعر، وتمارين فهم القراءة لطلاب المرحلة الثانوية (الصفوف 10-12).

## المميزات

- **المصادقة** - تسجيل المستخدمين وتسجيل الدخول بالبريد الإلكتروني/كلمة المرور، اختيار الصف، الموافقة على الشروط
- **فئات الاختبارات**:
  - النحو
  - الصرف
  - الأدب
  - الشعر
  - القراءة والنصوص
- **لوحة التحكم** - الأقسام الرئيسية: الرئيسية، المهام، لوحة المتصدرين، الملف الشخصي
- **نظام الاختبارات** - اختبارات تفاعلية بأنواع أسئلة متعددة، مؤقت، نتائج فورية
- **لوحة الإدارة** - إدارة الاختبارات للأساتذة/المديرين، محرر الأسئلة، إدارة النصوص، عرض النتائج
- **سجل النشاط** - تتبع الاختبارات المنجزة والأداء عبر الزمن
- **الإعدادات** - تخصيص السمة (فاتح/داكن/النظام)، الإشعارات، الخصوصية، وضع البطاطا
- **واجهة ثنائية اللغة** - واجهة عربية كاملة مع دعم RTL
- **ميزات الهاتف** - الإشعارات الفورية، مشاركة النتائج، التحديثات داخل التطبيق

## التقنيات

- **إطار العمل**: Flutter 3.x
- **اللغة**: Dart 3.x
- **إدارة الحالة**: Provider
- **التوجيه**: GoRouter
- **الخلفي**: Supabase (المصادقة، قاعدة البيانات)
- **الإعلانات**: Google Mobile Ads
- **البنية**: بنية نظيفة تعتمد على الميزات

## البدء

### المتطلبات

- Flutter SDK 3.x
- Dart 3.x
- حساب Supabase

### التثبيت

1. استنساخ المستودع:
   ```bash
   git clone https://github.com/hamzaelaiyat/ArabiLogia.git
   cd ArabiLogia
   ```

2. تثبيت التبعيات:
   ```bash
   flutter pub get
   ```

3. إعداد Supabase:
   - إنشاء مشروع Supabase
   - نسخ `.env.example` إلى `.env`:
     ```bash
     cp .env.example .env
     ```
   - تحديث `.env` ببيانات Supabase:
     ```
     SUPABASE_URL=https://your-project.supabase.co
     SUPABASE_ANON_KEY=your-anon-key
     ```

4. تشغيل التطبيق:
   ```bash
   flutter run
   ```

### البناء للإصدار

```bash
flutter build apk --release
flutter build ipa --release  # iOS (يتطلب macOS)
```

## هيكل المشروع

```
lib/
├── core/
│   ├── config/         # إعداد Supabase
│   ├── constants/      # سلاسل التطبيق، المسارات، المحتوى القانوني
│   ├── routes/         # توجيه التطبيق (GoRouter)
│   ├── services/       # خدمات API (Supabase، الإشعارات، إلخ)
│   ├── theme/          # إعداد السمة (فاتح/داكن/الرموز)
│   └── widgets/        # أدوات مشتركة (مكونات زجاجية، إعلانات أصلية)
├── features/
│   ├── admin/         # لوحة الإدارة (محرر الاختبارات، النتائج، الإعدادات)
│   ├── auth/          # تسجيل الدخول، التسجيل، نسيت كلمة المرور
│   ├── dashboard/      # التطبيق الرئيسي (الرئيسية، الاختبارات، المتصدرين، الملف الشخصي، السجل)
│   └── legal/         # المحتوى القانوني
├── providers/          # إدارة الحالة بـ Provider
└── main.dart          # نقطة دخول التطبيق
```
